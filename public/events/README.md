# Event Photos Directory

Place your event photos in subdirectories here:

```
events/
├── uipath-forward-2025/
│   ├── photo1.jpg
│   ├── photo2.jpg
│   └── photo3.jpg
├── poland-meetup-2024/
│   └── photo1.jpg
└── workshop-automation/
    └── photo1.jpg
```

**Note:** Photos will be accessible at `/PortfolioMZ/events/[folder]/[filename]`

For example:
- File: `events/uipath-forward-2025/photo1.jpg`
- URL: `https://mikzielinski.github.io/PortfolioMZ/events/uipath-forward-2025/photo1.jpg`
